The Amazing World of CSS

CSS (Cascading Style Sheets) allows you to create great-looking web pages. Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

After reading Chapter 3 of CSS the missing manual I am starting to feel pretty comfortable with using CSS for styling web pages. The first concept that requires grasping is the selector and decleration pattern that is the core of all style declerations. From there it is important to understand what our selectors are targeting. Selectors can be as sepcific as a single element to a broad range of elements on a page. Understanding type, class, Id, and attribute selectors are some of the key concepts involved here. Finally understand child, adjacent, and pseudo selectors will help round out the knowledge needed to effectively use CSS to style web pages.
